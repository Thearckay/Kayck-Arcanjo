> "Olá, mundo!" — Este é o repositório da minha primeira landing page aplicada, desenvolvida para centralizar minhas informações e colocar em prática estudos de desenvolvimento web.

---

## Sobre o Projeto

Este projeto foi o meu ponto de partida no desenvolvimento web com **foco em responsividade**. Ele funciona como um portfólio online onde apresento minha trajetória em Análise e Desenvolvimento de Sistemas, minhas competências e formas de contato.

### Objetivos alcançados:
* **Prática de HTML5 & CSS3:** Aplicação de estruturas semânticas e estilização personalizada.
* **Design Responsivo:** Desenvolvimento de uma interface que se adapta tanto para Desktop quanto para Mobile.
* **Apresentação Profissional:** Seção dedicada a skills, currículo e certificações.

---
